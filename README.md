# 🎓 GATE AI Professor Pro
**An Intelligent AI Tutor for Engineering Students (GATE & University Exams)**

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_svg)](https://aigateproject-tmvp3m5jws7smk3jkd2nn4.streamlit.app/)

## 🚀 Overview
GATE AI Professor Pro is a full-stack AI application designed to help engineering students master complex subjects like **Operating Systems, Computer Networks, and AI**. Built with a "Learning in Public" philosophy, this tool simplifies technical jargon into easy-to-understand analogies.

## ✨ Key Features
* **AI-Powered Tutoring:** Uses the `Llama-3.3-70b` model via **Groq Cloud** for lightning-fast responses.
* **Syllabus Focused:** Specifically tuned to handle GATE-level technical queries.
* **Clean UI:** Built with **Streamlit** for a seamless, responsive user experience.
* **Professional Deployment:** Hosted on **Streamlit Cloud** with secure environment variable management.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **AI Engine:** Groq API (Llama 3.3 Versatile)
* **Framework:** Streamlit
* **Version Control:** Git & GitHub
